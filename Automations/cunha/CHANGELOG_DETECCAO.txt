COMO USAR
---------
Simplesmente execute o script normalmente:

    python Automations/cunha/cunhaLabV2.py

O script detectará automaticamente a máquina e usará as configurações corretas.

Para personalizar threads:

    python Automations/cunha/cunhaLabV2.py --threads=24
